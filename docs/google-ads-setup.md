## Recommandations de démarrage

1. **Lancer avec correspondance exacte** `[mot-clé]` uniquement — le budget de 7$/jour est serré, éviter le broad match qui gaspille.
2. **Conversion tracking** : configurer l'événement de conversion sur la soumission du formulaire `/#formulaire` avant le lancement.
3. **Page de destination** : faire atterrir les groupes 1 et 2 sur `/#formulaire` (ancre directe formulaire), groupe 3 sur une version avec mention OBNL si possible.
4. **CPC max suggéré** : 1,50–2,50 $ CAD pour ces mots-clés (faible volume, peu de concurrence directe).
5. **Revue hebdo** : analyser Search Terms Report chaque semaine pour ajouter des négatifs et identifier de nouveaux mots-clés pertinents.
6. **Seuil de décision** : après 30 clics par groupe → conserver ou mettre en pause selon le taux de conversion.
